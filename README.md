# CantaTuHistoria.es - Músicas Personalizadas con IA (España) 🇪🇸

Plataforma completa de generación de canciones personalizadas de regalo para el mercado de España, optimizada con quiz interactivo, integración con IA y checkout sin comisiones vía Revolut & IBAN instantáneo.

## 🚀 Características
- **Página de Inicio Impactante:** Diseño moderno en modo oscuro con tonos púrpura/rosa (`#7C3AED` a `#EC4899`), badges de ocasiones y reproductor de ejemplos.
- **Quiz Interactivo en 9 Pasos:** Captura nombres, ocasión, estilo musical, historia personal y frase única.
- **Previa de Canción & Simulador IA:** Generación en tiempo real de letras y previa musical.
- **Checkout Revolut & SEPA Instantáneo:** Cobro en Euros (€14,90) sin comisiones bancarias ni esperas de pasarelas.
- **Notificación por WhatsApp:** Redirección automática para enviar el comprobante y liberar el enlace HD.

## 📦 Despliegue 100% Gratuito
1. **GitHub Pages:**
   - Ve a los Ajustes de este repositorio en GitHub -> `Pages`.
   - Selecciona la rama `main` y guarda.
   - Tu sitio estará online en: `https://llellenenen-droid.github.io/cancio-personalizada-espanha/`

2. **Vercel / Netlify:**
   - Importa este repositorio desde tu cuenta de Vercel (gratis) para obtener SSL automático y dominio personalizado.
